## 🎯 연구 목적

2025학년도부터 전면 시행되는 **2022 개정 교육과정(고교학점제)** 하에서 고등학생들의 **과목 선택, 평가 제도, 교육환경**에 대한 인식을 조사하고, **Iloh의 생태학적 진로 모형**(정보·시간·기회)과 **유데모니아(Eudaimonia) 웰빙** 관점을 통합하여 학생 경험을 종합 분석합니다.

---

## 📊 설문 구조 (5-Part, 27문항)

```
PART 1. 기본 정보 (4문항)
   └─ 학년, 학교 유형, 지역, 과목 선택 경험

PART 2. 과목 선택 및 교육과정 (Q1~Q6)
   ├─ 과목 선택 요인, 개설 충분성, 안내 제공
   ├─ 진로 연계, 선택 어려움 (다중체크)
   └─ 개선 의견 (개방형)

PART 3. 평가 및 학점 이수 (Q7~Q12)
   ├─ 성취평가제 이해·공정성
   ├─ 5등급 병기, 미이수 제도
   └─ 평가 문제점·개선 (개방형)

PART 4. 정보 · 시간 · 기회 — Iloh 모형 (Q13~Q20)
   ├─ 📡 정보: 진로 정보 접근성·질·획득 경로
   ├─ ⏳ 시간: 학습 시간 자율성·의미
   ├─ 🚪 기회: 교육 기회 접근·공정성
   └─ 가장 큰 장벽 선택

PART 5. 웰빙(유데모니아) · 종합 (Q21~Q27)
   ├─ 🌱 자기 성장 실감
   ├─ 💪 자기결정감 (자율성)
   ├─ 🔥 몰입 경험 (Flow)
   ├─ 🤝 관계·소속감
   ├─ ✨ 삶의 목적·의미
   ├─ 전반적 만족도
   └─ 자유 의견 (개방형)
```
